# Presuntas estafas en Guadalajara, Jalisco — (FICCIÓN)

Este repositorio contiene una página creada como material de ficción para una novela. El contenido adopta el formato periodístico como recurso narrativo, pero **no** representa hechos reales ni personas verdaderas.

Archivos incluidos:
- `index.html` — Página principal (marcada como ficción).
- `press_release.md` — Nota de prensa ficticia para uso en la novela.
- `social_thread.md` — Hilo de redes sociales ficticio para promocionar la pieza dentro de la historia.

Cómo publicar en GitHub Pages:
1. Confirma que los archivos estén en la rama `main` (o en la rama que uses).
2. En el repositorio, ve a **Settings → Pages**.
3. En *Source*, selecciona la rama (`main`) y la carpeta `/ (root)`.
4. Guarda los cambios. GitHub generará una URL tipo `https://<tu-usuario>.github.io/<repo>/`.

Aviso legal:
- Este proyecto es una obra de ficción. Cualquier parecido con personas reales es pura coincidencia.
- Si planeas publicar material que se base en hechos reales, toma precauciones legales y evita acusaciones directas sin pruebas verificables.
